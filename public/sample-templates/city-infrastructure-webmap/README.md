# Interactive City Infrastructure Web Map

## Overview
A responsive web map showing city infrastructure with real-time data integration using Leaflet.js and modern web technologies.

## What's Included
- Complete HTML/CSS/JavaScript application
- Sample infrastructure data
- REST API integration examples
- Setup and deployment instructions
- Customization guide

## Requirements
- Modern web browser
- Web server (local or hosted)
- Basic knowledge of HTML/CSS/JavaScript
- Optional: Node.js for development server

## Quick Start
1. Extract files to web server directory
2. Open `index.html` in browser
3. Customize data sources in `config.js`
4. Modify styling in `styles.css`
5. Deploy to web hosting service

## Features
- Interactive Leaflet map
- Real-time data updates
- Layer toggle controls
- Popup information windows
- Mobile-responsive design
- Search functionality

## Data Integration
- GeoJSON data support
