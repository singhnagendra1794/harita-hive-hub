# Urban Flood Risk Mapping Template

## Overview
This template provides a complete workflow for creating flood risk maps using DEM data, rainfall patterns, and urban infrastructure analysis in QGIS.

## What's Included
- QGIS Project File (.qgz)
- Sample DEM Data (GeoTIFF format)
- Python Scripts for automation
- Step-by-Step Guide (PDF)
- Video Tutorial links

## Requirements
- QGIS 3.16 or higher
- Python 3.7+ with GDAL
- Minimum 4GB RAM
- 2GB free disk space

## Quick Start
1. Open `flood_risk_mapping.qgz` in QGIS
2. Run the processing scripts in order:
   - 01_prepare_dem.py
   - 02_calculate_flow.py
   - 03_risk_analysis.py
3. Follow the detailed guide for customization

## Data Sources
- Sample DEM: 30m resolution elevation data
- Rainfall data: Historical precipitation records
- Infrastructure: Sample urban features

## Output
- Flood risk zones (shapefile)
- Risk assessment maps (PDF)
- Flow direction rasters
- Statistical reports
