# HaritaHive Project Templates

This folder contains sample project template files that will be uploaded to Supabase Storage.

## Templates Available:

1. **Urban Flood Risk Mapping** - QGIS project for flood risk analysis
2. **Land Use Classification with Sentinel-2** - Python ML project for land use classification
3. **Interactive City Infrastructure Web Map** - Web mapping application
4. **NDVI Crop Health Monitoring Dashboard** - Agricultural monitoring dashboard
5. **Urban Growth Analysis with QGIS** - Urban planning analysis
6. **Air Quality Monitoring Network Design** - Environmental monitoring optimization

## File Structure:

Each template includes:
- Complete source code/project files
- Sample datasets
- Step-by-step documentation
- Setup and installation instructions
- Video tutorials (where applicable)

## Usage:

1. Download the template ZIP file
2. Extract to your local directory
3. Follow the README instructions in each template
4. Customize for your specific use case
