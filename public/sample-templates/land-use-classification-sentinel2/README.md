# Land Use Classification with Sentinel-2

## Overview
Automated land use classification using machine learning and Sentinel-2 satellite imagery through Google Earth Engine.

## What's Included
- Jupyter Notebooks with complete workflow
- Pre-trained machine learning models
- Training and validation datasets
- Complete documentation
- Video walkthrough

## Requirements
- Python 3.8+
- Google Earth Engine account
- Required packages: earthengine-api, scikit-learn, pandas, matplotlib
- Minimum 8GB RAM
- Internet connection for GEE access

## Quick Start
1. Set up Google Earth Engine authentication
2. Install required packages: `pip install -r requirements.txt`
3. Open `land_use_classification.ipynb`
4. Run cells sequentially
5. Customize for your area of interest

## Features
- Multi-temporal Sentinel-2 analysis
- Random Forest classification
- Accuracy assessment
- Time series analysis
- Export capabilities

## Output
- Land use classification maps
- Accuracy metrics and confusion matrix
- Feature importance analysis
- Exportable GeoTIFF files

## Model Performance
- Overall Accuracy: 89.5%
- Kappa Coefficient: 0.87
- Classes: Urban, Forest, Agriculture, Water, Barren
